  Anatomy of a URL:
  A URL (Uniform Resource Locator) is the web address that identifies a resource on the internet. 
  It consists of several components, including the protocol (e.g., HTTP or HTTPS), domain name, path, and query parameters. 
  Understanding the structure of a URL is important for web development, 
  as it's the basis for making requests to web servers and handling routing in your applications.

      This code handles a GET request for the "welcomeMessage" 
      route and extracts the "user" query parameter.
      It uses the "getWelcomeMessage" function to generate a welcome message 
      for the specified user and sends it as a response.
      
      // script for greating
      app.get("/welcomeMessage", (req, res) => {
          const clientName = req.query.user;
          const welcomeMessage = getWelcomeMessage(clientName);
          res.send({ data: welcomeMessage })
      });
